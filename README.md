# Wiebe Consulting v1.0 - Sports PT Revenue & Retention System

Modern, professional website built for Wiebe Consulting - specialized in helping owner-led sports & ortho PT clinics maximize revenue from existing patients.

## Tech Stack

- **Framework:** Next.js 15 with React 19 & TypeScript
- **Styling:** Tailwind CSS 3.4
- **Animations:** Framer Motion 11
- **Icons:** Lucide React
- **Fonts:** Inter & Space Grotesk (Google Fonts)

## Features

- ✅ Fully responsive design (mobile-first)
- ✅ Professional futuristic aesthetic optimized for consulting/healthcare
- ✅ Smooth scroll animations with Framer Motion
- ✅ SEO optimized with proper meta tags
- ✅ Fast performance with Next.js 15 App Router
- ✅ TypeScript for type safety
- ✅ Gradient effects and modern UI elements

## Sections

1. **Hero** - Main value proposition with CTA
2. **Problem** - The profit leak in Sports PT
3. **System** - 3-pillar Revenue & Retention System
4. **Process** - 60-Day Sprint breakdown
5. **Qualification** - Who this is for (and not for)
6. **Pricing** - Founding cohort offer + standard program
7. **Requirements** - What's needed from clients
8. **About** - Company positioning
9. **FAQ** - Common questions
10. **CTA** - Final conversion section
11. **Footer** - Branding and legal

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3005](http://localhost:3005) in your browser.

## Color Scheme

- **Primary:** Blue (#0ea5e9) - Trust, professionalism
- **Accent:** Purple (#a855f7) - Innovation, premium feel
